Get Started

Build and run AtomicOS in minutes with our streamlined development environment

# Clone the repository
git clone https://github.com/ipenas-cl/AtomicOS.git
cd AtomicOS

# Build the operating system
make all

# Run in QEMU emulator
make run

# Run test suite
make test
